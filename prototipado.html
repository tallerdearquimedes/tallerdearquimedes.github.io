Prototipado

Del diseño a la evidencia

En este espacio se construyen objetos para responder preguntas. El prototipo no es el resultado final, es el medio para entender, validar y mejorar.

¿Qué significa prototipar?

Prototipar no es fabricar piezas.

Es construir artefactos con un propósito claro: responder una pregunta técnica, validar una idea o evidenciar un fenómeno.

Cada prototipo existe porque hay algo que se quiere entender, no solo porque algo se puede fabricar.

Metodología

1. Planteamiento

Se define una pregunta o problema concreto.

2. Hipótesis

Se establece una idea inicial que será puesta a prueba.

3. Diseño

Se traduce la idea en un objeto: geometría, materiales y proceso.

4. Construcción

Se fabrica el prototipo, enfrentando decisiones reales.

5. Prueba

Se observa, mide o interactúa con el sistema.

6. Evidencia

Se obtiene un resultado tangible.

7. Iteración

Se ajusta el diseño con base en lo aprendido.

Proyectos en desarrollo

El laboratorio es el espacio donde las ideas aún no terminan de ser, pero ya pueden probarse. Aquí los proyectos no aparecen como piezas cerradas, sino como etapas visibles de una búsqueda técnica en desarrollo.

Proyecto 01

Motor de Inducción Primario

Génesis experimental · Electromagnetismo
Este dispositivo representa el punto de ruptura entre la teoría y la acción. Lo construí cuando tenía 12 años, es un ejercicio de arqueología técnica que utiliza el embobinado manual para domesticar el campo magnético. Más que un motor, es la prueba física de la conversión de energía eléctrica en movimiento rotatorio mediante fuerza electromotriz.
Fundamento: Interacción de campos magnéticos y torque mecánico.
Estado: Prototipo funcional.
Proyecto 02

Caldera

Sistema térmodinámico · Generación de presión
Un sistema diseñado para el estudio del cambio de fase y la energía cinética del vapor. Esta caldera no es un elemento aislado, sino el núcleo de potencia para un ecosistema mecánico mayor. Explora la termodinámica aplicada, el control de fluidos y la resistencia de materiales bajo estrés de presión constante.
Fundamento: Transferencia de calor, leyes de los gases y mecánica de fluidos.
Estado: Prototipo funcional.
Proyecto 03

Autómata cinático: Colibrí

Cinemática de precisión · Transmisión de movimiento
Este modelo es una exploración de la cinemática aplicada. Utiliza un sistema de biela-manivela y un paralelogramo articulado para traducir la rotación pura en una trayectoria orgánica compleja. Aquí, el movimiento deja de ser una abstracción matemática para convertirse en un sistema geométrico tangible que prioriza la relación entre torque y velocidad.
Fundamento: Engranajes, mecanismos de cuatro barras y síntesis de trayectoria.
Estado: Modelo demostrativo funcional.
Proyecto 04

Línea de producción automatizada

Integración de sistemas · Automatización y Robótica
Una arquitectura de automatización industrial a escala que integra control digital y ejecución mecánica. El sistema coordina una banda transportadora con manipuladores de 4 y 6 grados de libertad (DOF), gestionados mediante microcontroladores. Representa el flujo de operaciones autónomas y la armonización de múltiples variables de control en un entorno de producción.
Fundamento: Robótica de servicios, lógica de control (Arduino/PLC) y cinemática inversa.
Estado: Sistema complejo en desarrollo.

¿Cómo se construyen los prototipos?

Los prototipos se desarrollan mediante la integración de distintas tecnologías:

Corte láser, router CNC e impresión 3D. Cada herramienta se utiliza no por disponibilidad, sino por pertinencia.

¿Qué se obtiene?

El resultado no es solo un objeto.

Es evidencia: comportamiento, medición, interacción.

Un prototipo permite entender, validar y mejorar.
